# Contact Management System

Flask app for verifying contact information


### Required Tools

- **Docker** (v20.10+)
- **kubectl** (v1.25+)
- **Kind** (v0.17+) or **Minikube** (v1.28+)
- **Python** (v3.11+) - for local development
- **PostgreSQL Client** (psql) - for database access


```bash
chmod +x deploy.sh
./deploy.sh
```
